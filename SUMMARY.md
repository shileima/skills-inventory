## 🚀 使用建议

### 推荐工作流

**1. 日常使用**
```
列出所有技能             # 查看概览
搜索 xxx 技能            # 快速查找
打开 xxx 技能            # 查看详情
```

**2. 技能管理**
```
搜索 test 技能           # 找出测试技能
删除 test-xxx 技能       # 清理测试技能
```

**3. 定期维护**
```
列出所有技能             # 检查总数
ls -la ~/.trash/skills/  # 清理旧备份
```

### 最佳实践

1. **删除前备份** - 始终选择备份选项
2. **定期清理** - 每月清理一次 `~/.trash/skills/`
3. **避免重复** - 使用符号链接代替复制
4. **保留原版** - 不要删除原始技能
